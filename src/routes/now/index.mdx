## Positronic

I'm having a blast working on [Positronic](https://github.com/sfioritto/positronic). Positronic is a framework for building LLM powered automations driven by a CLI. Right now I'm using it to:

* Present the low value emails in my inboxes in a UI that lets me process them super quickly e.g. it plucks action items from the millions, perhaps billions of daily emails that my children's schools generate for me. One click archiving of all marketing emails, view bills and receipts on one page, etc.
* Summarize the work my team has been doing all week per dev and slack me
* Automatically match receipts with expenses in Mercury
* Scan often for emails that need my attention immediately
* Build a Hacker News digest that's tailored for me

Mainly I use it as an excuse to avoid developer rust and hone my skills using coding agents, (Claude Code all day every day), as I'm not programming as much at work these days.
